## 运行

确保相关开发环境及代码编辑器正确配置

-   终端运行

    -   进入项目根目录
    -   键入 flutter pub get 安装插件
    -   键入 flutter run 编译&运行项目至模拟器

-   编辑器运行 - Android studio
    -   安装 dart&flutter 相关插件
    -   点击顶部工具栏 绿色按钮（确保 main.dart 显示为 flutter logo）
-   编辑器运行 - VSCode
    -   安装 dart&flutter 相关插件
    -   打开 lib -> main.dart 文件
    -   确保底部状态栏显示正确的设备，点击顶部工具栏下箭头 -> Start Debugging
